main help

There are three high level modes:

OUTLINE, DATABASE and EDITOR.

TAB cycles between OUTLINE and DATABASE modes.

SHIFT_TAB cycles between OUTLINE and SEARCH modes.

:[e]dit puts you in EDITOR mode and quiting EDITOR mode with either :q or :x puts you in OUTLINE mode.

OUTLINE and EDITOR mode support the following somewhat Vim-like modes:

  • NORMAL
  • INSERT
  • COMMAND_LINE
  • VISUAL_LINE //not available in OUTLINE
  • VISUAL_BLOCK //not available in OUTLINE
  • VISUAL
  • REPLACE
  • DATABASE //not available in Editor
  • FILE_DISPLAY //not available in Editor
  • NO_ROWS -> only editing possible is inserting a new row
  • SEARCH -> shows titles and notes with highlighting
